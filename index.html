About each element


Element 1 contains the follwing: basic shapes, lighting, textures, shadows and motion.

Element 2 contains the following: mesh manipulation, heigh map, enhancing enviroment textures, continued use of lighting and motion

Element 3 contains: moveable player mesh, player animation and player interaction/physics.

Element 4 contains: use of gui to create menu system, use of audio, transition from menu to scene, menu interaction and music/sound effect in response to scene events or actions.
